Eine neue Klassifikation anlegen

Sollte für ein Projekt noch keine Klassifikation existieren (andernfalls kann diese importiert werden, siehe unten), so kann mit dem Klassifikations-Editor eine neue erstellt werden. Um eine neue Klassifikation anzulegen, muss zunächst der Projektname (Kurzname oder Abkürzung) im Feld "‘Neuer Projektname"’ eingetragen werden. Danach kann in "‘Basiert auf"’ die Klassifikationskonfiguration ausgewählt werden, auf der die neue Klassifikation basieren soll. Anschließend wird mit einem Klick auf "‘Neu"’ die neue Klassifikationskonfiguration erstellt.
