Datenabgleich

Das kollaborative Arbeiten innerhalb eines Projektes wird durch den Datenabglich ermöglicht. Mithilfe des Abgleiches können die einzelnen von jedem Benutzer lokal - d.h. an seinem Rechner eingegebenen - vorhandenen Daten in das zentrale Repositorium gespielt werden und so anderen Benutzern zugänglich gemacht werden. Ebenso kommen andererseits dem einspielenden Benutzer alle für ihn neuen Daten aus dem Repositorium - d.h. diejenigen Daten oder Änderungen von Daten, die in seiner lokalen Version noch nicht vorhanden sind - zu. Der Ablauf eines Datenabgleiches, zu starten unter unter ▹Datenbank▹AlleDaten abgleichen, vollzieht sich dabei automatisch, es sollte allerdings darauf geachtet werden, dass immer nur ein Benutzer Daten mit dem Repositorium abgleicht und nicht mehrere zum selben Zeitpunkt. Sollten Fehler auftreten, so ist eventuell eine falsche Repositoriums-URL (siehe Kapitel 6.1.2) dafür verantwortlich oder aber es liegt z.B. eine Störung beim Repositoriumserver vor. Für weitere Hilfestellungen siehe Kapitel "‘kommt noch"’ (unten).
