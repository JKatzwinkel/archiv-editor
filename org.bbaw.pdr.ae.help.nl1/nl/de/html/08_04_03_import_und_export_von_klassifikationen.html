Import und Export von Klassifikationen

Um Klassifikationen in Dateien zu exportieren und somit zum Import auf anderen Rechnern verfügbar zu machen gibt es unter ▹Einstellungen▹Sicherungder Klassifikationen erstellen... eine entsprechende Funktion. Im folgenden muss ein Zielverzeichnis für die Sicherungsdateien festgelegt werden und die Durchführung der Sicherung mittels "‘Ok"’ bestätigt werden, woraufhin sämtliche im Editor vorhandenen Klassifikationen aller Projekte in Unterordner samt Information über den Sicherungszeitpunkt im angegebenen Verzeichnis gesichert werden. Dabei wird für jedes Projekt, welches im Editor eine Klassifikation besitzt, im Ordner "‘config"’ eine nach diesem benannte Datei erzeugt, die alle Informationen über Markups und semantische Klassifikationen beinhaltet.

Für das Einspielen zuvor exportierter Klassifikationen gibt es zwei Varianten. Einerseits können einzelne Klassifikationen von Projekten zu den sich bereits im Editor befindlichen hinzugefügt oder ersetzt werden. Um dies durchzuführen, muss unter ▹Datei▹Importieren... als Importtyp "‘Markup-Konfiguration"’ gewählt werden und im nächsten Fenster über "‘Auswählen"’ die konkrete Datei (für gewöhnlich benannt nach dem betreffenden Projekt) benannt und über "‘Datei laden"’ ausgewählt werden. Im letzten Schritt kann festgelegt werden, ob - sofern für dieses Projekt bereits eine Klassifikation im Editor vorliegt - die vorhandene Klassifikation überschrieben oder die neueinzuspielende umbenannt werden soll.



Die zweite Variante bietet die Möglichkeit einen kompletten Satz gesicherter Klassifikationen einzuspielen, wobei sämtliche (!) bereits im Editor vorhandene Klassifikationen automatisch entfernt werden. Dieser Vorgang lässt sich über ▹Einstellungen▹Sicherungder Klassifikationen laden... starten. Es folgt die Warnung, dass alle im Editor vorzufindenen Klassifikationen zuerst gelöscht und durch die neu eingespielten ersetzt werden. Anschließend muss als Quellverzeichnis der Ordner, indem die exportierten Sicherungen vorliegen gewählt werden. Dessen Name sollte für gewöhnlich mit "‘AEConfig"’ beginnen und darüber hinaus Datum und Uhrzeit der Sicherung tragen. Wichtig bei dem Quellverzeichnis ist, dass dieses ein "‘config"’-Unterverzeichnis beherbergt, indem zwar die Klassifikationen liegen, aber das für den Import gewählte Verzeichnis, das darüberliegende ist (siehe Bild).
