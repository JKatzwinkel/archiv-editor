Quellentypen bearbeiten

Im Quellentypen-Editor sind am linken Rand alle vorhandenen Quellentypen gelistet. Ganze Quellentypen lassen sich oben unter "‘Neuer Quellentyp"’ durch Eingabe eines Bezeichners und den "‘Neu"’-Button anlegen. Gelöscht werden angewählte Quellentypen über entsprechenden Button unten.

Ist ein Typ angewählt, so können im Quellenformular die Formularfelder einer Quelle ausgewählt werden. Eine sprachspezifische Beschreibung des Quellentyps kann unter "‘Beschreibung"’ abgelegt werden. Um den Typ einer Quelle zu visualisieren kann ihm außerdem ein Symbol zugeordnet werden.

Je nach Quellentyp stellt der Quellen-Editor verschiedene Formularfelder zur Verfügung. Es sollte mindestens ein Name oder ein Titel eingegeben werden (in der Regel ist dies das erste Datenfeld). Weitere Felder (z.B. Untertitel, Verlag, Auflage) lassen sich im Quellenformular über den Button "‘Felder hinzufügen"’ generieren. Soll ein Quellentyp beim Arbeiten mit dem Archiv-Editor nicht mit angezeigt werden, so wird unter "‘Ausblenden"’ oben rechts ein Häkchen gesetzt.
