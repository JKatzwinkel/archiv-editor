Allgemeine Einstellungen

Neben den grundlegenden Konfigurationen, wie z.B. den Spracheinstellungen, lassen sich im Archiv-Editor viele speziell auf den Benutzer zugeschnittene Einstellungen wie z.B. das Definieren von Shortcuts (Tastaturkürzeln) oder Darstellungsoptionen bis hin zur Farbe von Markups vornehmen. Über ▹Einstellungen▹Einstellungen lässt sich eine Maske, in der entsprechende Angaben geändert werden können, öffnen. Im folgenden werden die wichtigsten zur Wahl stehenden Punkte kurz erklärt, wobei auf ausführliche Erläuterungen aufgrund der weitestgehend für sich sprechenden Einstellungsoptionen verzichtet wird. Den Updates ist dabei später ein eigenes Unterkapitel gewidmet.

Allgemein

Neben einer Übersicht über die Repositoriumseinstellungen, wie z.B. Name und Repositoriums-URL, können die Einstellungen zur Sprache, das automatische Einloggen des Benutzers beim Programmstart und die automatischen Updates (siehe dazu auch Kapitel 8.5.1) (de)aktiviert werden. Unter ▹Tastenkombinationen können zudem Shortcuts verändert werden, um jeden Benutzer durch die individuelle Angepassung von Tastaturkürzeln ein schnelleres Arbeiten mit dem Archiv-Editor zu ermöglichen.

Aspekt

Hier können die verwendeten Schemata für die einzelnen Klassifikationen konfiguriert werden. Im Allgemeinen sollten dies die Projektklassifikationen sein. Ferner können unter ▹Ansicht die Anzeige eines einzelnen Aspektes inklusive der für bestimmte Markuptypen verwendeten Farbe editiert werden. Unter ▹Facettierte Aspektsuche und ▹Markup-Favoriten können Such- bzw. Markupvorschläge festgelegt werden.

Person

Neben wenigen Optionen zur Darstellung von Personenobjekten, finden sich auch hier unter ▹FacettiertePersonensuche die Möglichkeit bei der Personen-Suche vorzuschlagende Markups anzugeben.

Quellen-Einstellungen

Die Quellen-Einstellungen beinhalten Darstellungsoptionen der Quellenobjekte, wobei hauptsächlich für den Darstellungsnamen zu verwendende Elemente gewählt werden können und dazu die Einordnung in der Baum-Ansicht modifiziert werden kann.
