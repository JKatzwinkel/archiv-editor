Eine neue Quelle anlegen

Klicken Sie auf "‘Neue Quelle anlegen"’.Es öffnet sich der Quellen-Editor.

Jetzt wählen Sie einen Quellentyp aus. Der Quellentyp bestimmt, welche Felder anfangs zur Verfügung stehen. Sollten die Felder nicht ausreichen, können Sie weitere hinzufügen.

Wählen Sie zum Beispiel "‘Buch"’ und geben Sie unter "‘Titel"’ einen beliebigen Buchtitel ein. Mit einem Klick auf "‘Speichern"’ können Sie die neue Quelle abspeichern.
