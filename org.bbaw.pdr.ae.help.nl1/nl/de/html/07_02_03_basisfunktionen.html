Basisfunktionen

(a) Generale

Semantische Klassifikation

Weisen Sie dem Aspekt zunächst eine Klassifikation zu. Damit werden im Archiv-Editor Aspekte in Kategorien eingeordnet. Jedes Projekt hat sein eigenes Klassifikationsschema.

Der erste Aspekt einer Person sollte aus praktischen Gründen als Normname klassifiziert werden. Wählen Sie dazu aus der Liste den Eintrag "‘NormName_DE"’ aus. Wenn Sie eine Person mit dem Zauberstab erstellt haben, ist dieser Eintrag bereits ausgewählt.

Üblicherweise werden Sie Ihr eigenes Klassifikationsschema verwenden. Sie können aber auch die Schemata anderer Projekte benutzen. Es ist auch möglich, mehrere Klassifikationen anzugeben.

Inhalt Hier können Sie im unteren Textfeld "‘Inhalt des Aspekts"’ eine neue Personeninformation eingeben. Diese kann aus einem Beschreibungstext oder aus einem Quelltext bestehen, z.B "‘Der Komponist Georg Friedrich Händel wurde 1685 in Halle geboren"’.

Der erste Aspekt, den Sie eingeben, sollte ein Name sein. Namen dienen zur Identifizierung von Personen und werden zur Anzeige in der Personenliste des Archiv-Editors verwendet. Geben Sie dazu einfach den bevorzugten Namen der neu angelegten Person in das Textfeld ein.

Markup

Um den Inhalt des Textes systematisch durchsuchbar zu machen, sollte dieser mit Markup versehen werden.

Markieren Sie dazu ein einzelnes Wort oder einen Abschnitt des Notifikation-Textes. Wählen Sie nun aus den Markup-Listen in der Mitte der Seite aus, um was für einen Inhalt es sich bei dem markierten Text handelt.

Es sollten mindestens "‘Element"’ und "‘Type"’ ausgewählt werden. Weitere Angaben sind optional und werden nicht immer benötigt. Für die Zuweisung klicken Sie auf die Schaltfläche "‘Zuweisen"’ über dem Notification-Text. Das Markup des Textes erscheint nun farblich hervorgehoben.

Beim Anlegen eines Normnamens sollte mindestens der Nachname der Person mit Markup versehen werden, damit die Person leicht in der alphabetischen Liste auffindbar ist.

(b) Quellen

Jeder Aspekt benötigt eine Quellenangabe. Sie können dazu entweder aus den vorhandenen Quellen auswählen oder eine neue Quelle erstellen.

Wenn Sie eine neue Quelle erstellen, wird diese nach dem Speichern automatisch für diesen Aspekt als Quelle ausgewählt.

Falls die Information aus mehreren Quellen hervorgeht, ist es möglich, weitere Quellen anzugeben, indem man auf "‘mehr Quellen"’ klickt.

Intern

Das Feld "‘Intern"’ dient zur Identifikation einer Stelle innerhalb einer Quelle, z.B. einer Seitenzahl oder eines Aktenblatts, falls diese nicht mithilfe des Quellen-Editors erfasst werden kann.

Zuverl ä ssigkeit

Jeder Quellenangabe muss außerdem ein Zuverlässigkeitswert zugewiesen werden. Die Zuverlässigkeit meint eine Bewertung der Information hinsichtlich ihrer Sicherheit und Belegbarkeit. Zur Auswahl stehen "‘certain"’, "‘probable"’ und "‘unsure"’. Die Bewertung bezieht sich nicht auf die gesamte Quelle, sondern lediglich auf den aktuellen Aspekt.

certain = sicher, eindeutig belegt, anerkannt

probable = wahrscheinlich, anzunehmen, meistens anerkannt, nur schwach belegt

uncertain = unsicher, vermutet, spekulativ, meistens nicht anerkannt, nur theoretisch möglich

Interpretation

Hier können Hinweise zur Interpretation des Aspekt-Inhaltes eingegeben werden. Dies kann verwendet werden, wenn die Deutung nicht klar aus dem Text hervorgeht. Die Länge ist auf 1.000 Zeichen begrenzt.
