Automatische Updates

Sofern nicht anders konfiguriert, überprüft der Archiv-Editor beim Start, ob eine neue Version vorhanden ist. Sollte dies der Fall sein, so erscheint während des Ladevorgangs ein Sicherheitshinweis, der auf die bevorstehende Installation hinweist und Sie zur Bestätigung der Installation von nichtzertifizierten Komponenten auffordert. Sollten Sie Bedenken haben oder aus anderen Gründen ein Update vermeiden wollen, so können Sie an dieser Stelle abbrechen. Andernfalls wird das Update installiert, wobei gegebenenfalls der Firewall die Installation bestätigt werden muss.
