Kollaboratives Arbeiten

Der Archiv-Editor ist eine Komponente einer umfassenden Architektur, welche das kollaborative Arbeiten in mehreren Projekten und in Projektverbünden unterstützt.

Mehrere Mitarbeiter eines Projektes können parallel zueinander am gleichen Datenbestand arbeiten, welcher in einem zentralen Repositorium gespeichert wird. Die Daten können auf Knopfdruck zwischen Archiv-Editor und Repositorium abgeglichen werden.

Im zentralen Repositorium ist für jedes Projekt ein eigener Arbeitsbereich definiert. Dadurch können andere Projekte die gleiche Infrastruktur nutzen, ohne dass Interferenzen befürchtet werden müssen. Jedes einzelne Projekt kann seine Daten individuell präsentieren - oder auch im Verbund mit anderen Datenbeständen.
