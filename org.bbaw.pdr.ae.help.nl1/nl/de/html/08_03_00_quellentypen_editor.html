Quellentypen-Editor

Quellen werden mithilfe von vordefinierten Quellentypen erfasst (z.B. Buch, Zeitschrift, Artikel). Vorhandene Quellentypen können im Quellentypen-Editor bearbeitet oder ausgeblendet werden, ebenso wie neue Quellentypen erstellt werden können. Der Quellentypen-Editor wird über ▹Einstellungen▹Quellentypen-Editoröffnen geöffnet.
