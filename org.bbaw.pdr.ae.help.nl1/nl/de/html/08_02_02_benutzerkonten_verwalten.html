Benutzerkonten verwalten

Jeder Bearbeiter, der ein eigenes Konto besitzt, kann seine Zugangsdaten und nähere Informationen zur eigenen Person bearbeiten. Unter ▹Einstellungen▹Benutzerdatenbearbeiten lässt sich - nach Eingabe der aktuellen Zugangsdaten - eine Maske aufrufen, in welcher Benutzernamen, Passwort, nähere Angaben zum Bearbeiter und Kontaktdaten geändert werden können.

Benutzer, die über die Berechtigungen eines Admins verfügen, haben zudem die Möglichkeit mit Hilfe eines speziellen Werkzeuges unter

http://XXXXXXXXXXXXXXXXXXXXXXXXX.XX

sämtliche Benutzerdaten ihres Projektes zu bearbeiten, d.h. neu anlegen, verändern und löschen. Dieses Werkzeug läuft komplett selbstständig und setzt daher keine Installation des Archiv-Editors voraus. Ebenso kann es von jedem internetfähigen Rechner benutzt werden; von Nöten sind lediglich die Zugangsdaten eines Administratoraccounts, mit welchen man sich zuallererst unter obiger Adresse anmelden muss. Anschließend führt das Symbol, welches eine Person abbildet zu einer - ähnlich der im Archiv-Editor vorhandenen - Baumansicht, in der das Projekt auszuwählen ist, dessen Benutzer bearbeiten werden sollen. Dazu muss in der Baumansicht auf den kleinen Pfeil ganz links vom Projektnamen geklickt werden, woraufhin sich unter diesem eine Liste mit sämtlichen Benutzern des Projektes ausklappt. Ganz am Anfang der Liste befindet sich die Option "‘New User"’, welche es ermöglicht neue Benutzerkonten zu erstellen. Dabei muss in der sich rechts von der Baumansicht öffnenden Maske ein Benutzername samt Passwort und Zugangsberechtigungen festgelegt werden. Zudem können nähere Angaben z.B. Kontaktdaten zum Benutzer gemacht werden.

Soll ein bereits bestehender Benutzer bearbeitet werden, so muss lediglich auf dessen Name in der Baum-Ansicht geklickt werden und anschließend können sämtliche Informationen in selbiger Maske bearbeiten werden. Um sich wieder auszuloggen, muss in der kleinen Iconleiste am oberen Fensterrand das rote Icon mit dem Ausschaltknopf angeklickt werden.
