Programm starten

Zum Starten des Programms öffnen Sie die ausführbare Datei "‘Archiv-Editor.exe"’. Das Programm startet mit einem Ladebildschirm. Sollte es das erste Mal sein, dass das Programm gestartet wird, so erscheint ein Installationsdialog. Hier können Sie sich für den Advanced- oder Lite-Modus entscheiden (siehe: kommt noch) und das Standardverzeichnis des Archiv-Editors festlegen. Sollten Sie vorhaben, den Ort des Installationsverzeichnises des Editors zu verlegen, so entfernen Sie einfach das zweite Häckchen. Nachdem bestätigen durch "‘Speichern"’ erscheint eine Eingabemaske, welche die Einstellungen des Repositoriums abfragt. Dazu muss der Rechner mit dem Internet verbunden sein. In der Eingabemaske müssen die Identifikationsnummer und Webadresse des Repositoriums, sowie die Projektnummer angegeben werden. Sollten diese Identifikatorn nicht bekannt sein, so kann der Editor nur im lokalen Modus benutzt werden und sämtliche eingegebene Daten können später nicht für das Projekt im Repositorium benutzt werden.Desweiteren sei daraufhingewiesen, dass der Archiv-Editor beim Starten automatisch nach Updates, d.h. aktuelleren Versionen sucht und gegebenenfalls eine Sicherheitsabfrage erscheint, welche bestätigt werden muss, um die Updates zu installieren.
