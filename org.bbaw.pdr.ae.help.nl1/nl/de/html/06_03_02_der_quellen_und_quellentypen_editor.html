Der Quellen- und Quellentypen-Editor

Soll eine neue Quelle (z.B. über ▹Quelle▹NeueQuelle) angelegt werden, so öffnet sich der Quellen-Editor. Hier wird ein Quellentyp gewählt und über ▸Neu eine Maske für die zu diesem Typ relevanten Eckdaten erzeugt. Welche vordefinierten Quellentypen zur Wahl stehen und wie die dazugehörigen Eingabemasken aussehen, kann mittels des Quellentypen-Editor (zu finden unter ▹Einstellungen▹Quellentypen-Editoröffnen) festgelegt werden. Über der Maske befinden sich ähnlich wie im Personen-Editor die administrativen Daten, wie die PDR-ID, Ersteller und Erstellungsdatum der Quelle im Editor.

Für jede bereits angelegte Quelle kann ebenfalls der Quellen-Editor aufgerufen werden. Dazu muss diese über die Baum-Ansicht geöffnet werden und im Menü ▹Quelle▹Quellebearbeiten... angewählt werden.
