User

Der User (Benutzer) hat die gleichen Rechte wie der Gast, d.h. Betrachten aller sich im Editor befindlichen Daten, darf jedoch zusätzlich eigene Informationen, z.B. in Form von Personen oder Aspekten, hinzufügen. Diese Daten (und nur diese!) dürfen auch nachträglich bearbeitet oder gelöscht werden, sofern diese noch nicht mittels eines Abgleiches in das Repositorium gespielt wurden (s.o). Ebenso dürfen keine Veränderungen an den semantischen Klassifikationen und Markups vorgenommen werden.
