Benutzer

Der Archiv-Editor wurde speziell auf das parallele Arbeiten mehrerer Bearbeiter innerhalb eines Pojektes ausgelegt. Daher ist es notwendig, dass jeder Bearbeiter ein eigenes Benutzerkonto zugewiesen bekommt, welches sich in seinen Benutzungsrechten von anderen Benutzern unterscheiden kann. Zudem lassen sich mit Hilfe von Benutzerkonten die in den Editor eingegebenen Informationen immer dem jeweiligen Bearbeiter zuordnen.
