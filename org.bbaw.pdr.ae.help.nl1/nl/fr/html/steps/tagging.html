Was ist Tagging?


Tagging ist ein mark-up-Verfahren, bei dem einzelne Wörter oder Informationen innerhalb eines Textes systematisch gekennzeichnet werden. Dadurch lässt sich z.B. ein Name als Name im Text kennzeichnen und ist für maschinelle Suche später als Name auffindbar.

Je mehr Informationen und je genauer diese in einem Text getagt werden, desto exakter und treffsicherer lassen sich die Informationen aus dem Text durch Suchmaschinen wiederfinden.

Was kann und soll getagt werden?


Das Personendaten-Repositorium stellt ein eignens für das Tagging von Aspekten entwickeltes Tagging-Schema bereit, das auf den Erfahrungen mit TEI aufbaut.

Folgende fünf Grundkategorien:

  • persName - Name einer Person
  • placeName - Name eines Ortes
  • orgName - Name einer Organisation/Institution
  • name - Name eines Musikinstrumentes, Berufes, Werkes
  • date - Datumsangabe

Diese fünf Hauptelemente werden nach Type, Subtype, Role und Key näher gefasst. Die Klassifikationen für Type, Subtype und Role können an die spezifischen Forschungfelder angepasst werden.
