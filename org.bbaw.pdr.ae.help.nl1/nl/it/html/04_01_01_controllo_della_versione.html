Controllo della versione

a Controllare online

Si raccomanda fare l’aggiornamento attraverso un sito web. Lì, viene controllato automaticamente se il sistema contiene l’ultima versione. L’indirizzo è:

http://www.java.com/de/download/installed.jsp

b Controllare offline

Se il computer non dispone attualmente di una connessione a Internet, il metodo dipende dal sistema.

Windows: Fare clic su Start > Impostazioni > Pannello di controllo e scelga "‘Java"’. Nel pannello di controllo Java, selezionare la voce "‘Java"’ e clicca su "‘Mostra"’. Si apre un elenco delle versioni di Java installate.

Macintosh: Inviare le utilità o premare la scorciatoia "‘cmd + shift + u"’. Apre l’applicazione "‘Terminal"’. Nella finestra del terminale entra "‘java -version"’. La versione attualmente installata di Java è visualizzato.
