Installazione di Java Runtime Environment

L’Archiv-Editor richiede Java Runtime Environment (JRE) di Sun Microsystems. Pertanto, verifichi innanzitutto se il JRE è già installato. È già il caso nella maggior parte dei sistemi. Si assicuri di avere una versione adatta. Il programma richiede Java 6 ed almeno la versione 1.6.0.
