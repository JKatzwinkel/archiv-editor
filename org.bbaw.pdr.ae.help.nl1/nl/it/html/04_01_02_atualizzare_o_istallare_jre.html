Atualizzare o istallare JRE

Se JRE non è presente, installarlo sul proprio sistema. Se JRE è disponibile solo in una versione precedente alla 1.6.0, aggiornare la versione di Java. Questo è fatto anche sul sito web di cui sopra:

http://www.java.com/de/download/installed.jsp
