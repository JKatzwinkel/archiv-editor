Copiare gli archivi del programma

Ora installare l’Archiv-Editor sul suo computer. Se si apre il pacchetto, verrà visualizzata una cartella denominata "‘Archiv-Editor"’. Copiare l’intera cartella sul suo computer, ad esempio, sul desktop o in una directory del programma. Si prega di notare che non è più possibile spostare il programma in una directory diversa dopo il primo utilizzo.
