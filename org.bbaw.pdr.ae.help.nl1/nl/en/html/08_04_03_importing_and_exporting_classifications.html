Importing and exporting classifications

There is a specific function in the Editor for importing or exporting classification data, thus making it available on other computers. Click on ▹Preferences▹WriteConfig Backup.... Next, choose where the data should be saved and click on "‘Ok"’. This will create a folder labeled "‘config"’ (in the chosen location) containing the classifications of all projects registered in that Editor, including the date of when the backup was made. This folder contains a file for each project and contains all the markup and semantic classification for that project.

There are two ways to access imported classifications. One way is by selecting individual classifications and adding them to the Editor. This is done by clicking ▹File▹Import... and then selecting "‘Markup-Configuration"’ as the Input Type. In the next window click "‘Select"’ and choose the files wanted (usually named after the project). Finish the import by clicking "‘load files"’. Lastly, it is possible to choose if classifications that have the same name as those that already exist in the Editor should be overwritten or if those being imported should be renamed.



The second way is to import directly an entire collection of classifications, although the classifications that already exist in the Editor will then be automatically deleted (!). This is possible through ▹Preferences▹LoadConfig Backup.... A warning will appear, stating that all classifications currently in the Editor will be deleted and replaced with those imported. Next, choose the folders where the imported classifications are found. The name of this folder usually begins with “AEConfig” and includes the date and time that the backup was made. It is important to note that the “AEConfig” folder contains a sub-folder called “config”, which contains the classifications of all projects registered in that Editor. Every project that has a classification will appear in the folder “config” and will contain all the information for that project regarding markups and semantic classification (see picture).
