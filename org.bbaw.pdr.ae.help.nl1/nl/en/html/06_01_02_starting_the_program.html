Starting the program

To start the program open the file named “Archiv-Editor.exe”. A temporary screen will appear as the program loads. If this is the first time that the program is being started, a installation dialog will appear. Here you can choose the advanced or lite version of the editor (see:...) and establish the standard location for the program. If you want to change where the Editor program is kept simply remove the second checkmark. Click “Save” to confirm. Next, a form will appear requesting information about the repository settings. To fill out this form the computer needs to be connected to the internet. It asks for information regarding the identification number and web address of the Repository, as well as the project number. If this information is not immediately available, then the Editor can only be used locally - and all data that is entered cannot later be used for the project that is connected to the repository.It should be noted that when the Archiv-Editor is opened it searches automatically for updates. This means that a security window will sometimes appear at the start of the program and should be confirmed in order to install the updates.
