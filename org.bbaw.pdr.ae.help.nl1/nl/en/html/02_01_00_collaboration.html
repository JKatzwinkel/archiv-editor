Collaboration

The Archiv-Editor is one component within a broader architecture, and supports collaboration between multiple projects.

Researchers from one project can work parallel on the same data inventory, which is then saved in a central Repository. Data between the Archiv-Editor and the Repository is exchanged with a single click of the mouse.

Every Project receives its own designated workplace within the central Repository. Projects can thus use the same infrastructure without having to worry about outside interference. Every project can display its data individually or in conjunction with others.
