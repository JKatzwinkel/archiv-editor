General preferences

In addition to basic configurations, such as language preferences, the Archiv-Editor provides a range of customizations, such as defining keyboard shortcuts, or viewing options that go as far as changing the color of the markup. Click on ▹Preferences▹Preferences to open up a mask, in which such changes can be made. In the following section the most important options will be briefly explained. Preferences for updates will be covered in a later section.

General

In addition to repository settings such as the name of the repository and the URL, it is also possible to adjust here the language preferences, to set up an automatic log-in at the start of the program, and to activate or deactivate automatic updates (for more about updates see chapter 8.5.1). Keyboard shortcuts can be changed under ▹Keys, this allows the user to determine the shortcuts needed to work most effectively in the Archiv-Editor.

Aspect

Here you can configure the different systems used for classification. Generally the project-classification should be used. Furthermore, under ▹View you can adjust the display of an aspect, including the color of the markup. Under ▹FacetedAspectSearch and ▹FavoriteMarkup you can adjust the markup terms a user can choose from.

Person

Besides offering options for the display of person-objects, clicking ▹FacetedPersonSearch allows you to enter the Markup options used when searching for a person.

Reference preferences

In the reference preferences the display of a reference can be adjusted, this is accomplished mainly through changing the name that is displayed for the reference, thus changing placement in the tree-view.
