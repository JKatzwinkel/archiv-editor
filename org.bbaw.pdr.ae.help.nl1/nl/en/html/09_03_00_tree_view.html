Tree-view

This section builds upon the general description of the tree-view found in chapter 6.2.1 and discusses further special functions. For information about research tools that are integrated in the tree-view, see chapter 10.

Similar to the aspect-view, it is possible to activate or deactivate filters in the tree-view. These filters are however, not for aspects, but for objects (such as persons or references). To use a filter, simply right click on an object and choose the option “Filter”. Next, select a filter from the choices that appear. After the filter has been selected, only the filtered objects will appear in the tree-view. Activated filters are shown with a black check mark in the list of filter options, a filter can be deactivated by clicking it a second time.

Regardless which perspective has been chosen for the viewing-area, it is possible to have multiple objects open at the same time, found under different tabs (similar to a web browser). If a user is creating an aspect, for example, and is often consulting multiple persons, these persons can simply be opened in multliple tabs, thus avoiding having to open and close the persons and navigating through the tree. To open a person in a new tab, simply right click on them in the tree and choose “Open aspects in new tab”.



Yet another function allows you to display all the references used for the aspects of a person. Right click on the person and select “Open references in new tab”. A tab will then open which looks similar to an aspect tab, but provides a list of the references used for that person.



Tabs can be dragged and dropped into a different sequence and closed by clicking on the “x”, similar to a web browser.
