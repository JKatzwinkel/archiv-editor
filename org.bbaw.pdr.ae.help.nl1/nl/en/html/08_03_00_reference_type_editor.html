Reference-type editor

References are created with the help of predefined reference types (for example, a book, newspaper or article). Already existing reference types can be edited or hidden with the reference-type editor. New reference types can also be created. To open the reference-type editor, click ▹Preferences▹OpenReference Genre Editor.
