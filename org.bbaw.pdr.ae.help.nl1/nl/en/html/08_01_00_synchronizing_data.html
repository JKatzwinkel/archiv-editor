Synchronizing data

Collaboration within a project is possible through the synchronization of data. This allows the individual user to update the data on their local computer to the central repository, thus making it available to other researchers on the project. In turn, the individual user receives on their local computer any new data that has been updated by others into the repository, or changes that have been made, etc. To synchronize the local data with the repository and vice versa, select ▹Database▹UpdateAll Data. It should be noted, that only one user at a time can update their data to the repository. If there is an error, it could be due to a incorrect repository URL (see chapter 6.1.2) or a problem with the repository server. For further help see chapter “kommt noch”.
