Downloading the Archiv-Editor

The most recent version of the Archiv-Editor can be found at the following link:

http://pdr.bbaw.de/ae/installation

The Archiv-Editor is currently available for Windows 32bit, Windows 64bit, Mac OS 32bit, Mac OS 64bit, Linux GTK+ 32bit und Linux GTK+ 64bit.
