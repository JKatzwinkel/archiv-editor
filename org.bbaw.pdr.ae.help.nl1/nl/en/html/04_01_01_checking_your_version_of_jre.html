Checking your version of JRE

a Checking online

A quick check online is the easiest way to see which version of JRE is installed on your computer. You can automatically run a test at the following website:

http://www.java.com/de/download/installed.jsp

b Checking offline

If your computer is currently offline, you can still check your current version of JRE:

Windows: click Start > Control Panel > Programs, and then open “Java”. A “Java Control Panel” should open. Choose the tab “Java” and click on “view”. A list should appear with the currently installed version of Java.

Macintosh: open the disk utility window or click on “Finder” through the key combination shift+cmd+u. Start the program “Terminal” and enter in the Terminal window “java-version”. The currently installed version of Java should now be shown.
