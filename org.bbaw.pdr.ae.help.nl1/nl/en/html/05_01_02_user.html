User

The User has the same rights as a Guest, they have access to all data in the Editor. The User can however also add data, such as a person or an aspect. The data (and only this data) which the user has entered themselves may also be changed or deleted. Only if the data has been updated to the Repository is this no longer possible. A User cannot make any changes to semantic classification and markup.
