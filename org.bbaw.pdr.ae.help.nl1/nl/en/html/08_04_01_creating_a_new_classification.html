Creating a new classification

If a project has not yet created any classifications, this can be done with the classification-editor (see below for information about importing classifications). To create a new classification, click “New” in the already opened classification-editor. The project name must be entered in the field “New Provider”. In the second field “Based on”, the classification configuration can be selected on which the new classification should be based. Lastly, by clicking “New”, the new classification configuration can been created.
