User accounts and rights

There are three different kinds of user accounts: Guest, User, or Admin. These three types will now be briefly explained. It should be noted that user rights and access to data only apply to one project. Data that has not yet been synchronized with the repository can be recognized through the third number string in the ID number of the data, which will start with a "‘1"’.

In addition to the three mentioned user types, there is a fourth type called “pdrAdmin”. This user type is for technical purposes and is not necessary for project use. It is for the technical administration of the Repository and allows access to multiple projects.

  1. Guest
  2. User
  3. Admin
