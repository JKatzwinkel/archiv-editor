Editing reference types

A list of currently available reference types is found on the right side of the reference-type editor. New reference types can be created by entering the name of the type in the text field labeled “New Genre” and clicking “New”. Reference types can be eliminated by pressing the delete button found towards the bottom of the window.

If a reference type is selected, it is possible to choose the text fields that are then available for users entering data about that kind of reference. There should be at least a field for a name or title (this is usually the first field). Further fields (such as subtitle, publisher, edition) can be generated by clicking the button “Add Field”. If there is a reference type that should not be shown when using the editor, then check the box under “Ignore” in the upper right corner of the window.
