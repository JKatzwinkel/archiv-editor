Setting up Java Runtime Environment (JRE)

The Archiv-Editor uses Java Runtime Environment from Sun Microsystems. Check to see if JRE is installed on your computer - this should be the case for most systems. Next, confirm that it is a correct version. At least Java 6, Version 1.6.0. is needed.
