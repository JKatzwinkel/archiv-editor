Reference editor and reference-type editor

A new reference can be created by opening the reference editor (for example, through ▹Reference▹NewReference). Here you can choose the type of reference and with ▹New a mask will appear with the relevant template for this type. Which types of references are available and how the entry mask looks like can be adjusted with the reference-type editor (which can be found under ▹Preferences▹OpenReference Genre Editor). Above the mask is administrative data similar to that found in the person-editor, such as the PDR-ID, creator of the reference, and date of creation.

The reference editor can also be opened for references that have already been created. The reference must be opened from the tree-view and then selected in the task menu ▹Reference▹ModifyReference....
