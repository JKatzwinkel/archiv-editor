Splitting up a longer text into multiple aspects: how many texts belong in a single aspect?

When working with biographical data it is often the case that a longer text needs to be entered into the Editor. The Archiv-Editor is primarily made for short texts, thus longer texts should be divided. The question is: how should this be done? Can the text simply be divided into sentences, or should there be a division based on context?

The data model of the aspects is made so that it functions best when every piece of information is found in a single, comprehensible context.

A typical case is very basic biographical information. Normally, such data is structured like the following:

(Aspect 1) Born on October 13thin Schivelbein, Pommern.

This aspect contains a date, two place names (settlement and region), and the description of an event (a birth). This information thus has a context of both time and place.

It is impractical when an aspect contains information that is contextually not related. For example in this case a job title, which does not have anything to do with a birth.

(Aspect 1) Born on October 13thin Schivelbein, Pommern; doctor atCharitéhospital in Berlin.

If more than one context exists in an aspect, this creates an unnatural connection between the data. The goal of the data model is to document single contexts so that these are systematically searchable. Therefore, data related to different contexts should be divided into different aspects.

This is also applicable the other way around. When directly related data is split into different aspects, information can be lost. Information should not be broken down like the following:

(Aspect 1) worked as a doctor.

(Aspect 2) worked at Charitéhospital.

(Aspect 3) worked in Berlin.

The three aspects describe an act (1) as a profession, (2) in an institution, (3) at a place. Although individually they are fully valid aspects, the information belongs to the same context (“worked as a doctor at Charité hospital in Berlin”), and should be found within one aspect, not split between three:

(Aspect 1) worked as a doctorat Charitéhospital in Berlin.

The only exception to this rule is when single pieces of data are provided,. such as someone’s confession, gender or nationality. In this case it is sufficient to enter simply a single word in the aspect.

(Aspect 1) manly

(Aspect 2) protestant

(Aspect 3) German

These kinds of aspects can also be extended:

(Aspect 1) protestant,from 1874 catholic
