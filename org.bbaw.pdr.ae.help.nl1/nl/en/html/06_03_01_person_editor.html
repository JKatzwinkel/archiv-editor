Person-editor

The person-editor is not only for creating person-objects, it is also allows persons to be connected to an outside identifier, such as a PND, as well as to be connected to other persons within the Editor. Connecting one person-object to another is especially helpful when checking to see if a person has been entered more than once in the Editor. When a person is selected, the aspects of any connected persons are also shown in the aspect-view.

In addition, the person-editor includes administrative data, such as the user who created the person, the creation date, the date it was last edited, and the PDR-ID. The window for the person-editor opens as soon as a new person is created (as long as the option “mit einem normalisierten Namen erstellen” has not been chosen, see chapter 7.1.1) or when a person is edited that already exits in the tree-view. Editing or creating a person is possible by clicking on the appropriate symbol in the icon list or by choosing ▹Person in the task menu.
