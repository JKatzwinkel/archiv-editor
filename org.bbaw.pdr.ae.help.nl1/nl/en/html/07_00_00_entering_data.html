Entering data

The user interface of the Archiv-Editor is made up of a main window, which shows the three essential elements of the data model: persons, aspects, and references.Information provided through references is documented through aspects. A reference can thus be the basis for multiple aspects. Aspects always have a corresponding person and a person can have as many aspects as necessary.

graphic: References Aspects Persons

It is practical to create a person together with the corresponding aspect that contains their name. A reference is also necessary. After this information has been entered, all further person data can be added.
