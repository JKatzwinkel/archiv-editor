Special functions

The following functions are generally used only in special cases or for specials needs.

(a) Edit a person

Click on the icon “Modify Current Person”. This opens the person-editor

Clicking the tab “Identifiers” allows you to give the person an external identification number.

The standard installation provides the following choices:

PND (Personennormdatei, Germany)LCCN (Library of Congress Control Number, USA)ICCU (Istituto Centrale per il Catalogo Unico, Italy)VIAF (Virtual Authority File, international)

It is best to look up the external ID number directly on the website of the institution, for example on the website of the Library of Congress.

As many ID numbers can be added as wanted. It is also possible to rate the reliability of an ID number.

You can combine two persons under “Concurrences”. This is practical when one person has accidentally been entered twice.

(b) Creating an anonymous person

Click the “New person” button under “Person” in the task menu. This opens the person-editor.

Click on “Save” and a new person will be created and will appear in the list of anonymous persons. The new person is marked with a gold star. Because this person does not yet have a name, it will be shown with an internal identification number.

(c) Deleting a person

Click on the “delete person...” button.
