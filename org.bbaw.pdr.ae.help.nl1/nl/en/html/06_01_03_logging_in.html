Logging in

After the program has loaded, a window will open requesting a username and password. The account used to log in must belong to a project that has been coordinated with this particular Editor. The log-in window can be set up so that the username of a specific account always appears, requiring only the password. It is also possible for one account to be logged into automatically at the start of the Editor.
