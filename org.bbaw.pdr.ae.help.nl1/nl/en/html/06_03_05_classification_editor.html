Classification-editor

Classifications are used to structure aspects, and thus the data in the Archiv-Editor. It is important to note the difference between semantic classification, relationships, and markup. Semantic classification categorizes the entire aspect, relationships establish connections between objects, and markup allows one to structure the content of an aspect. Every project working with the Archiv-Editor has their own customized classification system. The classification-editor is opened with ▹Preferences▹OpenConfig Editor and with this editor one can create, delete and edit classifications.
