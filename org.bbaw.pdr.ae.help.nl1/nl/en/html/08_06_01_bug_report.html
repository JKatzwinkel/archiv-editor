bug report

Many computer programs due to their complexity still contain errors even after their public release. If an error should occur while using the Archiv-Editor, such as the program crashing, incorrect display, or even typos, these can be documented under ▹Help▹ReportErrors. A form will open in which the error can be documented in detail. Click ▹SendReport and then fill out the the next form which requires some data about the user, the same kind that is used for accounts in email programs such as “Mozillas Thunderbird” or “Outlook Express”.

Alternatively, you can send an email to archiv-editor@bbaw.de
