Copying the program data

You can now install Archiv-Editor onto your computer. When you open the program packet, you will see a folder called “Archiv-Editor”. Copy this folder onto your computer. You could copy it, for example, onto your desktop or the program menu. Please note that after you start the program for the first time, you cannot later move this folder to another location.
