Reference editor and reference-type editor

A new reference can be created by opening the reference editor (for example, through ▹Reference▹NewReference). Here you can choose the type of reference and with ▹New a form will appear with the relevant template for this type. Which types of references are available and which text fields a form has can be adjusted with the reference-type editor (which can be found under ▹Preferences▹OpenReference Genre Editor). At the top of the window is administrative data similar to that found in the person-editor, such as the PDR-ID, the creator of the reference, and the date of creation.

The reference editor can also be opened for references that have already been created. Open the reference in the tree-view and then select in the task menu ▹Reference▹ModifyReference....
