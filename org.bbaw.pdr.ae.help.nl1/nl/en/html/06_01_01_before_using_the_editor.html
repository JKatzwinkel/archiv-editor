Before using the Editor

Before you can start using the Archiv-Editor it needs to be clarified which repository the project will be using and, if there are multiple projects involved, how they will be connected. Each project within a repository is identified with a number, and the repositories themselves are also numbered. A repository needs to have its own web address in which the data can be exchanged; this means that the entire data of the project and all its user accounts are located at this web address. In addition, the user accounts need to be fully established, it is not sufficient to only create the user types mentioned above. This can be discussed with an administrative personnel from the Repository and will then be confirmed in the Repository itself.
