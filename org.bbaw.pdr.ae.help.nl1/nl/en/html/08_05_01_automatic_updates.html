Automatic updates

The Archiv-Editor checks if there is a new version available at every start of the program, as long as this hasnâ€™t been changed in the preferences. If there is a new version available, a warning appears while the program is loading which notifies the user of the installation and asks for a confirmation. If you have concerns about this update or want to skip it, you can cancel it at this time. Otherwise the update will be installed. Depending on the Firewall, the installation might have to be confirmed a second time.
